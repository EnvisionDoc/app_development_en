Using Base Icon Tools
=================
The dynamic elements in the graph usually consist of base icon tools, such as dynamic data, button, curve, progress bar, and widget plug-ins. The base icon tools are important components of the monitoring pages. This section introduces how to insert base icon tools in the graph area.
